Welcome to UrKidsWorld School Rest API Server!

Built on Spring boot with Spring security using Jwt token.

End Points HTTP verb Request Details Authorized Role Header Results
/users/login POST Sample Json "username" :"User1", "password" : “Pass123" All users Header: [{"key":"Content-Type","value":"application/json","description":""}] Authenticates User
/users/registration POST Sample Json "firstName" : "Nicky", "lastName" :"David", "userName" :"NickyDavid123", "email" :"NickyD@abc.com", "roles" :["admin"], "password" : “Pass1234” All users Header: [{"key":"Content-Type","value":"application/json","description":""}] Registers new user
/students POST "studName":"Annie", "classNumber":"1B", "parentUserName":"User1", "scoreMath":23.0, "scoreScience":24.0, "scoreWriting":25.0, "scoreReading":20.0, "scoreLifeSkills":22.0 ADMIN Header: [{"key":"Content-Type","value":"application/json","description":""}, {"key":"Authorization","value":"Bearer eyJhbGciOiJSUzI....","description":""}] Creates new student
/classes POST Sample Json "classNumber":"1B", "teacherUserName":"JohnDavid123" ADMIN Header: [{"key":"Content-Type","value":"application/json","description":""}, {"key":"Authorization","value":"Bearer eyJhbGciOiJSUzI....","description":""}] Creates new Class
/classes/classdetails GET Param classname ADMIN OR TEACHER Header: [{"key":"Authorization","value":"Bearer eyJhbGciOiJSUzI....","description":""}] Fetches the details of specific class
/students/class?classnumber GET Param classnumber ADMIN OR TEACHER Header: [{"key":"Authorization","value":"Bearer eyJhbGciOiJSUzI....","description":""}] Fetches all students in a specific class
/students/children GET Param classNumber PARENT Header: [{"key":"Authorization","value":"Bearer eyJhbGciOiJSUzI....","description":""}] Fetches the children details of user
/classes GET ADMIN Header: [{"key":"Authorization","value":"Bearer eyJhbGciOiJSUzI....","description":""}] Fetches all the available classrooms in school
/classes/hw PATCH "homeWorkStr":"Todays Hw" TEACHER Header: [{"key":"Content-Type","value":"application/json","description":""}, {"key":"Authorization","value":"Bearer eyJhbGciOiJSUzI....","description":""}] Update latest homework of specific class
